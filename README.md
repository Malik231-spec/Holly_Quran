# 🌙 Al-Qur'an Al-Kareem

> A beautiful, modern, and responsive Holy Qur'an web application built with React.
> Read the Holy Qur'an with elegant Arabic typography, Urdu translation, audio recitation, dark mode, and a premium reading experience.

---

## ✨ Features

✅ Complete Holy Qur'an
✅ Beautiful Arabic Ayat Display
✅ Urdu Translation Support
✅ English Translation Support
✅ Audio Recitation for Every Ayah
✅ Play / Pause Controls
✅ Ayah Copy Facility
✅ Search Surah by Name
✅ Dark & Light Mode Toggle
✅ Modern Premium UI
✅ Smooth Reading Experience
✅ Responsive Design
✅ Auto Sidebar Close on Mobile
✅ Desktop Optimized Experience
✅ Elegant Mushaf Reading Mode
✅ Fast & Lightweight
✅ API Based Quran Data

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/quran-app.git
```

Go to project folder:

```bash
cd quran-app
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

---

# 📂 Build For Production

```bash
npm run build
```
